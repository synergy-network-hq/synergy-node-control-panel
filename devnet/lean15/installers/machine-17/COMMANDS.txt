Linux/macOS Commands
--------------------
# One-command installation + firewall + start
./install_and_start.sh

# Status
./nodectl.sh status

# Live logs
./nodectl.sh logs --follow

# Last logs
./nodectl.sh logs

# Restart
./nodectl.sh restart

# Stop
./nodectl.sh stop

# Node metadata/ports
./nodectl.sh info

Windows PowerShell Commands
---------------------------

# Node metadata/ports
